# Breast Cancer Classification using Hybrid CNN-Transformer Architecture

A deep learning framework for automated breast cancer image classification using a Hybrid CNN-Transformer model that combines the local feature extraction capabilities of Convolutional Neural Networks (CNNs) with the global contextual learning power of Vision Transformers.

This project also integrates Explainable AI (XAI) techniques to improve interpretability and clinical transparency in medical image analysis.

---

## Features

- Hybrid CNN + Transformer architecture
- Breast cancer image classification
- Deep learning training and evaluation pipeline
- Explainable AI (XAI) integration
- Grad-CAM visualization
- Attention map visualization
- Occlusion sensitivity mapping
- Image preprocessing and augmentation
- Confusion matrix and performance metrics
- GPU-compatible training

---

## Model Architecture

The proposed framework combines:

### CNN Backbone
- Extracts local spatial features
- Learns edges, textures, and tissue patterns

### Transformer Encoder
- Captures long-range dependencies
- Improves contextual understanding of breast tumor structures

The hybrid architecture improves feature representation and classification performance compared to standalone CNN models.
